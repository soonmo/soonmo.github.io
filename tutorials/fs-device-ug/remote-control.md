원격 제어판은 FutureSmart 버전 **3.4** 이상에서 부터 지원됨, 원격 제어판을 사용하기 위해서는 먼저 복합기의 관리자 암호가 설정되어 있어야 함

## 복합기 관리자 암호 설정

  * **보안** 탭 선택
  * **로컬 운영자 암호 설정** 
      * 새암호: admin
      * 암호를 확인하십시오: admin
